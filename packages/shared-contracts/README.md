# packages/shared-contracts — Shared Contracts

Cross-service API, enum, error, and field-exposure contracts.

**Contents:**
- `api/` — Response envelope and error contract documentation.
- `enums/` — Common cross-service enum definitions.
- `safe-fields/` — Learner-safe vs internal field exposure contract.

**Constraints:**
- Documentation and schema definitions only. No runtime business logic.
- Used by Backend API, Flutter Mobile, and Admin Dashboard.

See `docs/phase-1/repo-structure.md`.
