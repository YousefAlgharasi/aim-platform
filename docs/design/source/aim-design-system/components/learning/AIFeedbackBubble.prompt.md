AI tutor feedback message — a chat bubble with the AIM gradient avatar. The signature element for AI-generated guidance.

```jsx
<AIFeedbackBubble tone="praise">Great sentence! Your use of <b>present perfect</b> is spot on.</AIFeedbackBubble>
<AIFeedbackBubble tone="correction">Try "have been" instead of "has been" with <b>I</b>.</AIFeedbackBubble>
<AIFeedbackBubble typing />
```

Tones: `neutral`, `praise` (green), `correction` (amber). Set `typing` to show the streaming indicator while the AI responds. The bubble starts from the leading edge, so it mirrors correctly in RTL.
