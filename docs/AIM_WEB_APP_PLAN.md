# AIM Platform — Web App Plan (React)

> **Status:** Stub — React Web MVP pilot already completed as historical baseline.

## Context

A React Web MVP pilot was completed and serves as the historical baseline for the platform. This document captures the plan for the production web app, which extends and replaces the pilot.

## Core Principles

- Shared API contract with the Flutter mobile app.
- No adaptive logic on the client side.
- No AI provider keys in the browser.

## Key Screens (MVP)

1. Landing / Marketing page
2. Registration & Login
3. Placement Assessment
4. Dashboard
5. Active Lesson
6. Exercise views (per skill type)
7. Progress Review
8. Profile & Settings
