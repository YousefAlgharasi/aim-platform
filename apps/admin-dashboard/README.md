# apps/admin-dashboard — Internal Admin Dashboard

Phase 1 internal admin surface foundation.

**Phase 1 scope:** App shell, layout, routing foundation, API client, role menu placeholder, and module placeholders only.

**Constraints:**
- Phase 1 admin is internal foundation only, not a full institute management platform.
- Does not expose learner-private AIM internals to unauthorized roles.
- Backend remains final authorization authority.

See `docs/phase-1/repo-structure.md` and `docs/phase-1/system-foundation-charter.md`.
